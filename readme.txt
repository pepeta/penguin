「Tropical Penguin」
南国気分を満喫する昼のペンギンのテーマです。


・文字コードについて
このテーマのtheme.htmlはUTF-8で作られています。
XOOPS Cube本体をダウンロードしてデフォルトでインストールした場合の文字コードはEUCですので、
このまま利用すると文字化けが発生します。
そういう場合はtheme.htmlを削除して、同梱されているtheme_euc.htmlをtheme.htmlとリネームしてお使いください。
＃ホダ塾ディストリビューションのデフォルト文字コードはUTF-8ですのでそのまま使えます。


・ナビゲーションについて
ナビゲーションに表示されている文字とリンクはtheme.htmlの54行目から62行目に記述されています。
サイトに合せて適宜、表示する文字、モジュール名、リンク先を修正、または追加、削除してお使いください。
